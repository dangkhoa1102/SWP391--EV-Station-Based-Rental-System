# Legacy Vanilla JavaScript Files

This folder contains old vanilla JavaScript code from before the React migration.

**These files are NOT used in the current React application.**

## Contents

- `/components` - Old vanilla JS UI components (notification.js, searchUI.js)
- `/modules` - Old vanilla JS modules (auth.js, search.js, searchModule.js)
- `/services` - Old vanilla JS services (bookingService.js)
- `/utils` - Old vanilla JS utilities (constants.js, dateUtils.js, modalUtils.js)

## Current React Structure

The React application uses a different structure:
- React components: `/src/components/`
- React services: `/src/services/`
- React utilities: `/src/utils/`

## Status

These files are kept for reference only and may be deleted in the future.
If you need similar functionality, check the new React implementations in `/src/`.
